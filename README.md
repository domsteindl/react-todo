# React To-do App

A simple to-do application built with React, TypeScript, and Tailwind CSS.

## Features
- Add tasks
- Mark tasks as completed
- Delete tasks

## Tech Stack
- React
- TypeScript
- Tailwind CSS
- Vite

## How to Run

1. Install dependencies:
    ```bash
    npm install
    ```
2. Start development server:
    ```bash
   npm run dev 
   ```
3. Open on the provided localhost
